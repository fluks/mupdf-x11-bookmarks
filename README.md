DESCRIPTION
===========

Bookmarks for mupdf on x11 platforms.

Bookmarks are saved to ~/.mupdf_bookmarks.

USAGE
-----

Press 'B' to save a bookmark for the document. It is only saved to a file
when the document is reloaded or the program is quit.
